14 # Using it.
15 You should read the test file as a sample for usage.
16 ```cpp
17 vector<double> data{1.0, 1.0, 2.0, 3.0, 19.0, 0.5, 14.0};
18 histo::Histo<double> h_scott(data); // Default method to calculate Breaks.
19 size_t bins = 10;
20 auto breaks_with_bins = histo::GenerateBreaksFromRangeAndBins<double>(0.0, 20.0, bins);
21 double width = 1.0;
22 auto breaks_with_fixed_width = histo::GenerateBreaksFromRangeAndWidth<double>(0.0, 20.0, width);
23 histo::Histo<double> h_with_bins(data, breaks_with_bins);
24 histo::Histo<double> h_with_width(data, breaks_with_width);
25 ```
26 
27 Each histogram has public members: `bins`, `breaks`, `counts` and `range`.
28 We can fill the histogram with `FillCounts(data)`, called at constructor.
29 The data is not stored in the histogram.
30 
31 We can fill the bins with more data to an existing histogram.
32 ```cpp
33 vector<double> extra_data{7.0, 13.0};
34 h_with_bins.FillCounts(extra_data);
35 ```
36 
37 We can also normalize the histogram to get a probability density function from it.
38 
39 ```cpp
40 histo::Histo<double, unsigned int> regular_histo(data);
41 histo::Histo<double, double> normalized_histogram = histo::NormalizeByArea(regular_histo);
42 ```
43 
44 Optionally, we can use VTK (vtkChartXY) to visualize the histogram.
45 
46 ```cpp
47 vector<double> data{0.0, 1.0, 1.0,1.0, 2.0, 3.0, 5.0, 5.0, 8.0, 8.0, 12.0};
48 Histo<double> h(data, histo::GenerateBreaksFromRangeAndBins<double>(0.0,15.0, 5));
49 h.PrintBreaksAndCounts(std::cout);
50 h.name = "withJustData";
51 visualize_histo(h, vtkChart::LINE);
52 visualize_histo(h, vtkChart::BAR);
53 ```
