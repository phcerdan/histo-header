Help functions to manually creating breaks from input range (low, upper) and desired number of bins.

You can use this function to force breaks in the place you want. It is specially useful if you want to store integers. For example, image your data is unsigned, [0,...,10] You can choose breaks in a way that the middle of the bins are unsigned: GenerateBreaksFromRangeAndBins( min - 0.5, max + 0.5, max + 1) If you use PrintCenter, you will the expected histo.

Template Parameters
PRECIis the type of the breaks, should be greater or as precise as T.
Parameters
lowfirst value of breaks.
upperlast value of breaks.
binsnumber of divisions.
Returns
breaks vector with the frontier values of each bin.
