Detailed Description

template<typename PRECI = double, typename PRECI_INTEGER = unsigned long int>
struct histo::Histo< PRECI, PRECI_INTEGER >

Histogram inspired by R. Simple, no dependancies, header-only. It accepts different data values and precission.

Template Parameters
TInput data type.
PRECIIt should be equal to T, except when T is int.
PRECI_INTEGERint type for counts data member.
