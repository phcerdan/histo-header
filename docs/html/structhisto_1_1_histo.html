template<typename T, typename PRECI = double, typename PRECI_INTEGER = unsigned long int>
struct histo::Histo< T, PRECI, PRECI_INTEGER >

Histogram inspired by R. Simple, no dependancies, header-only. It accepts different data values and precission.
